# 공통 용어 사전

> 도메인을 가리지 않고 워크스페이스 전체에서 쓰이는 용어입니다.
> 도메인별 용어는 `wiki/{도메인}/glossary.md`를 참조하세요.

| 용어 | 설명 |
|------|------|
| Command Center | 이 워크스페이스. ontology + wiki + GitHub Pages 사이트 |
| ALLSP_STD | ALLSP 회계 ERP 표준 코드베이스 (Java + JSP + MyBatis + DHTMLX) |
| ontology/ | 도메인 개념·관계·코드 위치를 구조화한 탐색 맵 (YAML) |
| wiki/ | 도메인별 비즈니스 정책·의사결정·시나리오를 정리하는 디렉토리 (Markdown) |
| dialect | 3개 DB(MariaDB / Oracle / MSSQL) 방언. 매퍼가 sqlmap/maria, sqlmap/oracle, sqlmap/mssql로 분리 |
| 결재 워크플로우 | esaApp 연동 + 상태코드(공통코드 A61/A64/A68/A69) 기반의 상신→결재→확정/반려 흐름 |
| DefaultVO | 모든 VO가 상속하는 베이스. 세션 사용자/IP/프로그램ID를 자동 주입 |
| 분개유형 (JRNLZ_TY_CD) | 같은 예산과목에 대해 차변/대변 회계계정을 달리 잡기 위한 분류 코드 |
| 다중과목 화면 (Multi) | AccExp102BMulti / 120Multi / 106Multi 3종. 한 품의에 N개 과목·거래처 입력 지원 |
