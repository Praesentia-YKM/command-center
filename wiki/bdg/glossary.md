# 예산 도메인 용어 사전

| 용어 | 설명 |
|------|------|
| 회계연도 (ACC_YY) | 예산이 적용되는 1년 단위. 모든 BDG_* 테이블의 PK 1번 |
| 회계구분 (ACC_DIV) | 일반회계 / 특별회계 구분 |
| 예산차수 (BDG_BDGDGR) | 회계연도 안에서 본예산/추경을 번호로 관리하는 단위 |
| 본예산 | BDG_CPL_DIV='M'. 회계연도 시작 시점의 예산 |
| 추경 | BDG_CPL_DIV='S'. 본예산 이후 추가/조정 편성된 예산 |
| 정책사업 (BDG_MNG_CLS) | 사업 위계 최상위. 정책 단위로 묶인 사업 그룹 |
| 단위사업 (BDG_MNG_BIZ) | 사업 위계 중간층. 정책사업의 하위 |
| 세부사업 (BDG_DTL_BIZ) | 사업 위계 최하위. 지출이 직접 참조하는 단위 |
| 지출과목 (BDG_EBDG_SBJ) | 세출(나가는 돈) 과목. 목그룹(H) / 목(M) / 세목(S) 3계층 |
| 수입과목 (BDG_IBDG_SBJ) | 세입(들어오는 돈) 과목. 관(K) / 항(H) / 목(M) / 세목(S) 4계층 |
| 세출예산편성 (BDG_EBDG_CPL) | 차수+편성구분+세부사업+과목별 예산 트랜잭션 행. 지출의 차감 대상 |
| 편성구분 (CPL_DIV) | Y(요구) → J(조정) → H(확정) 3단계 진행 |
| MARK_ORD | 과목 트리 정렬키. 10자리, LVL별 2자리. 01부터 순차 증가 |
| SP_CVR_MARKORD_UPDATE | 세출 MARK_ORD 계산 SP |
| SP_CVR_MARKORD_UPDATE_IMP | 세입 MARK_ORD 계산 SP |
| 편성 잔액 | BDG_EBDG_CPL의 편성액 - 차감액. 지출 통제의 기준 |
